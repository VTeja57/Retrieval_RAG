RAG DOCUMENT ASSISTANT - Simple Instructions

1. Install Python 3.8+ and Ollama from https://ollama.ai/
2. Download models: ollama pull phi3:3.8b and ollama pull nomic-embed-text:v1.5
3. Install packages: pip install -r requirements.txt
4. Run application: streamlit run rag_streamlit.py
5. Upload a PDF/CSV/TXT file and ask questions about it
6. The system will answer based on your document content
